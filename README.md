# manga-web-app
Manga reader app in python

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/MasterGar1/manga-web-app
    ```
2. Navigate to the project directory:
    ```bash
    cd manga-web-app
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
4.Run the app:
    ```bash
    python main.py
    ```
